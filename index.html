Hi. I'm Allahverdi, also known as Net User.

I'm a DevOps & Infrastructure enthusiast with a Python background. I like building tools for self-hosting and automating deployments via systemd.

I created GhostProvider, a TUI that scans GitHub repos, analyzes deployability, and deploys services as systemd user-level units. Supports Python, Node.js, Go, Rust, and static sites.

My focus is on automation and local-first infrastructure.
